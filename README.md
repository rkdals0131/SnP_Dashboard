# S&P 500 투자 대시보드 📊

**S&P 500의 매크로 경제 분석, 확률 기반 예측(Fan Chart), 실시간 차트를 한 곳에서!**

이 대시보드는 금융 애널리스트와 투자자를 위한 올인원 솔루션입니다. 복잡한 설정 없이 웹 브라우저에서 바로 사용할 수 있습니다.

---

## ✨ 주요 기능

### 📈 **1. 매크로 시황 요약**
- **MacroScore**: 여러 경제지표를 종합한 실시간 시황 점수
- **경제지표 카드**: VIX, 금리, 유가, 금, 실업률, CPI 등 핵심 지표
- **레짐 분석**: 금융여건, 성장, 인플레이션 상태 자동 판단
- **AI 요약**: Gemini AI로 시장 상황을 한 줄로 요약

### 🎯 **2. S&P 500 Fan Chart (확률 예측)**
- **분위 회귀 기반 예측**: 1개월, 3개월, 6개월, 12개월 예측
- **확률 구름 (Fan Chart)**: 90%, 80%, 50% 신뢰구간 시각화
- **시나리오 분석**: Risk-On, Risk-Off, 연준 긴축, 인플레이션 등
- **불확실성 게이지**: 현재 시장 불확실성 측정
- **기여도 분석**: 어떤 지표가 예측에 가장 큰 영향을 미치는지 확인

### 📊 **3. 실시간 차트**
- **분봉/시봉 차트**: 1분, 5분, 15분, 30분, 1시간, 1일 단위 캔들스틱
- **기술적 지표**: 이동평균선(SMA), 볼린저 밴드, RSI, MACD
- **실시간 가격**: 현재가, 고가, 저가, 거래량
- **자동 새로고침**: 최신 데이터로 자동 업데이트

## 🚀 빠른 시작 (3단계!)

### 1️⃣ **설치**
```bash
# 가상환경 생성 (선택사항이지만 권장)
python3 -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate

# 필요한 패키지 설치
pip install -r requirements.txt
```

### 2️⃣ **API 키 설정**
루트 폴더에 `.env` 파일을 만들고 FRED API 키를 입력하세요:
```bash
FRED_API_KEY=여기에_발급받은_키_입력
```

> 💡 **FRED API 키 발급하기**: https://fred.stlouisfed.org/docs/api/api_key.html (무료, 1분이면 발급 가능!)

### 3️⃣ **대시보드 실행**
```bash
python -m src.cli app
```

브라우저에서 http://localhost:8501 을 열면 끝! 🎉

## 📖 탭별 기능 안내

### 탭 A: 매크로 시황 요약
현재 시장 상황을 30초 안에 파악할 수 있습니다.

**사용법:**
- 실시간 데이터는 5분마다 자동 캐싱
- "탭 A 데이터 새로고침" 버튼으로 즉시 업데이트 가능
- AI 요약을 보려면 `.env`에 `GEMINI_API_KEY` 추가

**주요 메트릭:**
- **MacroScore**: +1에 가까울수록 우호적, -1에 가까울수록 비우호적
- **Breadth**: 상위 50% 이상 지표 비율 (동조화 정도)
- **레짐 배지**: 금융여건(Easy/Tight), 성장(확장/둔화), 인플레이션(상승/하락)

### 탭 B: S&P 500 Fan Chart
S&P 500의 확률 기반 예측을 시각화합니다.

**사용법:**
1. 왼쪽 사이드바에서 **시나리오 쇼크 슬라이더** 조정
   - 10Y UST, 3M UST, VIX, WTI, Gold, NFCI, CPI
2. 또는 **사전 시나리오** 드롭다운에서 선택
   - Risk-On, Risk-Off, 연준 긴축, 인플레이션 급등, 성장 우려
3. **예측 기간** 선택 (1M, 3M, 6M, 12M)
4. Fan Chart, 불확실성 게이지, 기여도 차트 확인
5. **시나리오 비교 히트맵**을 열어 여러 시나리오 동시 비교

**해석 방법:**
- **진한 밴드** (50%): 중앙값 근처 확률 높음
- **중간 밴드** (80%): 약간 극단적인 시나리오
- **연한 밴드** (90%): 극단적 상황
- **불확실성 게이지 > 1.0**: 평소보다 변동성 높음

### 탭 C: 실시간 차트
프로페셔널 트레이더를 위한 실시간 차트입니다.

**사용법:**
1. 왼쪽 사이드바 **실시간 차트 설정**에서 선택
   - **데이터 간격**: 1분, 5분, 15분, 30분, 1시간, 1일
   - **기간**: 1일, 5일, 1개월, 3개월, 6개월, 1년, 2년, 5년
2. 차트 옵션 선택
   - 거래량 표시
   - 이동평균선 (SMA 20, 50)
   - 볼린저 밴드
   - 기술적 지표 (RSI, MACD)
3. 필요시 "차트 새로고침" 클릭

**참고:**
- 1분 데이터는 최근 7일간만 사용 가능 (yfinance 제한)
- 장 마감 후에는 당일 데이터 표시

## 🔧 문제 해결

### Q: "원천 데이터를 찾을 수 없습니다" 오류
**A**: 왼쪽 사이드바에서 **"🚀 초기 설정"** 버튼을 클릭하세요. 처음 실행 시 데이터를 다운로드해야 합니다.

### Q: "모델 스냅샷이 없습니다" 경고
**A**: **"🎯 모델 재학습"** 버튼을 클릭하여 모델을 학습하세요. 초기 설정을 하면 자동으로 학습됩니다.

### Q: FRED API 키 오류
**A**:
1. https://fred.stlouisfed.org/ 에서 무료 계정 생성
2. API 키 발급 (상단 메뉴 → My Account → API Keys)
3. 프로젝트 루트에 `.env` 파일 생성 후 `FRED_API_KEY=발급받은_키` 입력

### Q: 탭 A에서 데이터가 안 보여요
**A**:
1. 인터넷 연결 확인
2. FRED API 키 확인
3. 사이드바에서 "탭 A 데이터 새로고침" 클릭
4. 그래도 안 되면 **"📥 데이터 업데이트"** 클릭

### Q: 실시간 차트가 느려요
**A**:
- 기간을 줄이세요 (예: 2년 → 6개월)
- 데이터 간격을 늘리세요 (예: 1분 → 1시간)
- 기술적 지표 체크박스를 끄세요

### Q: Fan Chart 예측이 이상해요
**A**:
- 최신 데이터로 업데이트: **"📥 데이터 업데이트"** 클릭
- 모델 재학습: **"🎯 모델 재학습"** 클릭
- 시나리오 슬라이더를 0으로 리셋

## 🎓 고급 사용법 (CLI)

웹 UI가 아닌 명령줄에서 직접 제어하고 싶다면:

### 1. 데이터 수집
```bash
python -m src.cli ingest --start 2010-01-01
```

### 2. 피처 생성
```bash
python -m src.cli features --calendar-start 2010-01-01
```

### 3. 모델 학습
```bash
python -m src.cli train --horizons 1M,3M,6M,12M
```

### 4. 모델 검증
```bash
python -m src.cli validate --train-window 60 --test-window 12
```

### 5. 시스템 상태 확인
```bash
python -m src.cli status
```

**전체 명령어 및 옵션:**
```bash
python -m src.cli --help
```
